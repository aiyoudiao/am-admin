## 🚀 Quick start

### backend

```bash
cd backend
pnpm install

# 要安装 docker + docker-compose，或者直接安装 Docker Desktop
pnpm run init
```

### frontend

```bash
cd frontend
pnpm install
pnpm run dev
```
